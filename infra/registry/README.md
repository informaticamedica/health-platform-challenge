# Registry

Espacio para levantar un registro privado de paquetes (por ejemplo, Verdaccio) para el ecosistema interno.

## Proposito

- Publicar paquetes compartidos de forma controlada.
- Facilitar consumo interno fuera del workspace local cuando sea necesario.

## Alcance esperado

- Configuracion de servidor de registry.
- Politicas de autenticacion y scopes.
- Guia de `.npmrc`/`.pnpmrc` para publicacion y consumo.

## Estado

- Carpeta en fase de preparacion conceptual.
