# Skills fhir

Mapeo entre modelos internos y recursos FHIR.

## Skills disponibles

- `skills/fhir-mapper-library/SKILL.md`: construye utilidades reutilizables para transformar datos minimos de dominio en recursos FHIR estandar.

## Regla de libreria

- Todo helper nuevo debe exportarse en `src/index.ts` y compilar a `dist` para consumo desde otros proyectos.
