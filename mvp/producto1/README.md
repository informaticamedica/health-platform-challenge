# Producto1

Subproyecto MVP en construccion con separacion front/back.

## Estructura

- `producto1-front/`: app cliente que consume contratos y design system.
- `producto1-back/`: servicio backend que integra paquetes core.

## Estado actual

- Ambos paquetes tienen dependencias declaradas pero aun no exponen scripts de ejecucion propios.

## Recomendacion inmediata

- Definir scripts minimos (`dev`, `build`, `test`, `start`) en front y back para habilitar ciclo completo de desarrollo.
