# Skills del repositorio

Este directorio define skills internas para acelerar tareas repetibles.

- Alcance: convenciones globales, nomenclatura y checks de calidad.
- Cada submodulo mantiene su propia carpeta `.agents/` para especializacion.

## Orquestacion general (headless)

- Skill de entrada: `skills/platform-orchestrator/SKILL.md`.
- Comandos canonicos: `new:mvp`, `new:db`, `new:package`, `new:template`, `new:p2t`.
- Contratos y mapeo: `ops/skill-registry.json` y `ops/schemas/*`.
